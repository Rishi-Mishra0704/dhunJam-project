# 📖 [Dhun_Jam] <a name="about-project"></a>

> DhunJam is a music app that allows users to listen to music. It is built with Next.js, Typescript, React.js and Chart.js.

**[Dhun_Jam]**

### Tech Stack <a name="tech-stack"></a>

<details>

<ul>

 </ul>
</details>
